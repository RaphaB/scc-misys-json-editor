Edit your Json file

User guide:

This tool is very simple to use. First of all, click on the link above. Then, you'll have to choose the type of data you want to add to your Json file (thanks to the drop-down list). Afterwards, click on the blue button "Add field". When you finish your new Json file, register it by clicking on the green button called "Submit new Object". Your new Json file is now in your "Download" directory. Enjoy.
